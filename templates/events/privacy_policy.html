Privacy Policy

Effective date: March 30, 2026

FurConnect ("we", "us", "our") powers schedule browsing and event planning for Furlingame conventions. Most site features are available without logging in.

What we collect

  • Schedule preferences (saved filter, selected day, favorites) stored in browser localStorage
  • Scroll position / view state for a singly-loaded session
  • Optional panel/host submission data if a user_fill action is supported (name, role, description, contact email)
  • Basic analytics only if enabled (page views, no personally-identifying content)

No mandatory account data

Browsing and using the public schedule does NOT require a user account. If login is enabled, only minimal username/email is stored and passwords are hashed. If not used, no login data is stored.

How we use data

  • Display event schedules and conventions
  • Keep UI preferences and scroll position for convenience
  • Process panel submissions and event management if your team enables that workflow
  • Support app performance; prevent abuse

Cookies/local storage

We use localStorage (or WebView storage) for UI state. No tracking cookies are required by default.

Your rights

Contact info@furconnect.org to request deletion or correction of any stored submission info.

Third-party services

If you integrate analytics / CDN, data is handled by those providers, but we do not sell personal data.
